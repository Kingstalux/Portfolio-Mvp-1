## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- VS Code
- Chrome Browser
- Gitbash


### Usage
- To get local copy :
 ``` node
  git clone https://github.com/Kingstalux/Portfolio-Mvp-1
  ```
  ```node
   npm install
 ```
 - To open in browser using webpack server :
  ```node
   npm start
 ```

 ## Testing

To test this project you need to :
``` node
    npm test
```
